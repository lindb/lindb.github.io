# Metadata

当前集群元数据配置管理。

# Broker

在多机房/Region模式下，当前集群下已经注册的Broker集群。

注册相应的Broker配置信息。

# Logic Database

当前集群下逻辑数据库列表。

创新或者修改对应逻辑数据库配置信息。

# Storage

当前集群下已经注册的Storage集群，每个Storage在启动的时候会完成自注册，正常情况不需要用户手动进行注册。

当Storage自注册失败时，可以通过手动方式注册相应的Storage配置信息。

# Database

当前集群下数据库列表。

创新或者修改对应数据库配置信息。

# Explore

浏览当前集群ETCD中的各类型的配置信息，可以通过Compare按钮来对应内存中各状态机中的信息。

查看与内存状态机中信息的对比结果，是否存在状态机中的信息与配置不一致情况。
