# API

LinDB 提供统一的 HTTP Rest API 对外服务，目前版本为 v1 ，统一在 /api/v1/* 下。

本文档主要介绍 LinDB 对外提供服务的 API，不涉及 LinDB 集群内部通信相关的 API。
