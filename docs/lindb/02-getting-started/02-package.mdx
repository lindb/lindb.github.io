## Install LinDB

Select your platform:
<TabGroup items={["Linux","MacOS","Windows"]}>
	<TabList>
  	<Tab>Linux</Tab>
    <Tab>MacOS</Tab>
    <Tab>Windows</Tab>
  </TabList>
  <TabPanels>
    <TabPanel>
			1. Download the package:
				- [lindb-v0.3.1-linux-amd64.tar.gz](https://github.com/lindb/lindb/releases/download/v0.3.1/lindb-v0.3.1-linux-amd64.tar.gz)
				- [lindb-v0.3.1-linux-arm64.tar.gz](https://github.com/lindb/lindb/releases/download/v0.3.1/lindb-v0.3.1-linux-arm64.tar.gz)
			
			2. Uncompress
			```sh
		  tar -zxvf lindb-v0.3.1-linux-amd64.tar.gz
			```
		  3. Run LinDB
			```sh
			./lind standalone run
			```

		</TabPanel>
    <TabPanel>
			1. Download the package:
				- [lindb-v0.3.1-darwin-amd64.tar.gz](https://github.com/lindb/lindb/releases/download/v0.3.1/lindb-v0.3.1-darwin-amd64.tar.gz)
				- [lindb-v0.3.1-darwim-arm64.tar.gz](https://github.com/lindb/lindb/releases/download/v0.3.1/lindb-v0.3.1-darwin-arm64.tar.gz)
			
			2. Uncompress
			```sh
		  tar -zxvf lindb-v0.3.1-darwin-amd64.tar.gz
			```
		  3. Run LinDB
			```sh
			./lind standalone run
			```

		</TabPanel>
    <TabPanel>
			1. Download the package:

				[lindb-v0.3.1-windows-amd64.tar.gz](https://github.com/lindb/lindb/releases/download/v0.3.1/lindb-v0.3.1-windows-amd64.zip)
			
			2. Uncompress
			```sh
		  unzip lindb-v0.3.1-windows-amd64.zip
			```
		  3. Run LinDB
			```sh
			./lind standalone run
			```
		</TabPanel>
  </TabPanels>
</TabGroup>

## Verify
After the above installation are done and LinDB is already started, LinDB command-line interface **lind-cli** can be launched to access LinDB.

```bash
./lind-cli
Welcome to the LinDB.
Server version: 0.3.1
lin@localhost:9000>
lin@localhost:9000> show databases;
+-----------+
| Database  |
+-----------+
| _internal |
+-----------+
1 rows in sets (4.218862ms)
```
